    Our Stock Analysis

    Using the percent change columns of the economic data, volume, and percent gain of the current month as well as the percent gain of the previous month we created our training model. We tested for the “R/G” column for whether or not the next month would having a negative or positive gain. We then split our training and testing sets and scaled the data. Afterwards we used “RandomOverSampler” to resample the data in order to have equal amount of outcomes to fit the model. I chose to use a RandomForestClassification model because since stocks have a lot of variability and volatility in their price action history I wanted to create stronger models off of weak ones in order to better understand how stock prices would be affected by these economic factors. Using this process I was able to have an accuracy of 65% which is a 15% improvement form using this same model but with only the Open, Close, and Volume columns.

    For the second model I used the same process using the same columns for the training and testing sets. However with this model I using the dataset including the stocks from the QQQ index to train the model. I then tested this model on the QQQ data to see how well it predicted the gain based on the data from all of the stocks combined. The model had an accuracy of 69% and when predicting the QQQ data it had an accuracy of 68%. What says is given the same economic conditions and how each of these stocks within the QQQ index performed.
